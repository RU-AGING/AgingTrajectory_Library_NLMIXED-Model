## 📌 Overview
This training packet provides a **self-contained SAS-native framework** for group-based trajectory modeling (GBTM).  
It replaces dependence on `PROC TRAJ` and is suitable for **high-security environments** where DLLs cannot be installed.  

The framework supports **five distributions**:
- Censored Normal  
- Poisson  
- Zero-Inflated Poisson (ZIP)  
- Negative Binomial (NB)  
- Zero-Inflated Negative Binomial (ZINB)  

Key Features:
- Balanced AND unbalanced panels
- BIC, AIC, –2LL fit statistics
- Posterior-based metrics: Entropy, Average Posterior Probabilities (APPs)
- Automated best model selection
- One-click report export (RTF/PDF/Excel)
- Optional bootstrapped CIs (serial or parallel)

## 🚀 Getting Started in 5 Minutes

1. **Load the framework macros**

%include "01_Framework/gbtm_framework.sas";

2. **Prepare or simulate data**  
Option A: Convert long data → wide format  

%prep_gbtm_data(data=my_long, id=person_id, time=month,
outcome=measure, value=value, T=12,
out=base_file_srs);
Option B: Simulate toy data  

%sim_data(dist=zinb, class=2, n=150, T=12, seed=101);

3. **Run trajectory models**  

%run_gbtm(data=base_file_srs, dist=zinb, max_classes=3, order=2, T=12);

4. **Export formatted report**  

%export_report(outfile="C:\results\gbtm_report.pdf", dest=pdf);
